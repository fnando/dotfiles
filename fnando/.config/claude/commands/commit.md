---
description: Commit changes
---

If project has a `Makefile`, before you commit, make sure to:

- run `make fmt` if available
- run `make check` if available

Once everything is ready, you can commit your changes. Commit with a 1-line
commit message up to 62 chars, using imperative mode (e.g. "Fix crash when not
able to expand global dir.").

- Avoid technical terms when possible
- Avoid multiple sentences
- **DO NOT** add "Co-Authored-By" lines
- **DO NOT** add a description
- End with a period
